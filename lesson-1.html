Почему важна семантика?

Без семантики страница — как книга без глав. Правильные теги помогают доступности и поиску.

Правило: на странице один <main>.

Памятка по семантическим тегам

  • <header> — шапка страницы/секции (логотип, поиск, навигация).
  • <nav> — меню навигации (набор ссылок <a>).
  • <main> — основной контент страницы. Один раз на страницу.
  • <section> — смысловой раздел с заголовком.
  • <article> — самостоятельная единица (пост, карточка, комментарий).
  • <aside> — боковая панель: доп. инфо, ссылки по теме.
  • <footer> — подвал страницы/секции (копирайт, контакты).

Иерархия заголовков

<h1> — главный заголовок; затем вложенно <h2>, <h3>… Не «прыгаем» через уровни без причины.

Навигация списком (понятно и для скринридеров)

<header>
  <nav aria-label="Главное меню">
    <ul>
      <li><a href="#">Домой</a></li>
      <li><a href="#">Портфолио</a></li>
      <li><a href="#">Контакты</a></li>
    </ul>
  </nav>
</header>

Изображения и alt — как писать

  • Зачем: текст-замена для тех, кто не видит картинку или если файл не загрузился.
  • Как: коротко и по смыслу роли изображения.
    • Аватар: alt="Фотография Кирилла"
    • Логотип: alt="Логотип студии Pixel Lab"
    • Иконка «Поиск»: alt="Поиск"
  • alt пустой для декора: alt="".
  • Избегаем: «картинка», «фото_123.jpg», повтор ближайшего заголовка.

Мини-шпаргалка для «Визитки»

<h1>Привет, мир!</h1>
<nav aria-label="Главное меню">
  <a href="#">Домой</a>
  <a href="#">Портфолио</a>
  <a href="#">Контакты</a>
</nav>
<img src="https://placekitten.com/240/160" alt="Фото автора">

Проверка на запоминание

Верно или неверно: на странице может быть только один <main>.
Что выбрать для карточки блога в списке постов?
Какая landmark-роль соответствует боковой панели (<aside>)?

Проект: мини-визитка

  1. <h1> «Привет, мир!»
  2. <nav> с 3 ссылками
  3. <img alt="...">
Редактор решения
Самостоятельно: добавьте секцию «Навыки» со списком и подписи ссылкам.

Дополнительные идеи

  • Оберните картинку в <figure> + <figcaption>.
  • Добавьте блок «Контакты» внизу.
