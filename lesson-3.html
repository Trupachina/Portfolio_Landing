Видимый фокус — навигация для клавиатуры

Не отключайте outline без замены: настройте :focus-visible. Иконкам-кнопкам — осмысленный aria-label. Цвета держим контрастными.

Мини-правило: контраст текста к фону ≥ 4.5:1.

Памятка

  • Контраст текста к фону ≥ 4.5:1 (обычный текст) и ≥ 3:1 (крупный ≥ 24px или 18.5px жирный).
  • Фокус: :focus-visible с заметной обводкой и outline-offset.
  • Иконкам-кнопкам добавляйте aria-label — доступное имя («Открыть меню», «Поиск»).

Шаблон видимого фокуса

.btn:focus{ outline:3px solid #007aff; outline-offset:2px; }
.btn:focus:not(:focus-visible){ outline:none; }

Шпаргалка по контрасту

/* Избегайте #9aa3b2 на #fff — слабоват контраст */
.good-text{ color:#0b0f14; background:#f6f8ff; } /* ок для UI */

Как скрыть фокус у мыши, но оставить его для клавиатуры?
Как дать доступное имя иконке-кнопке?

Проект: доступные кнопки

  1. Сделайте фокус видимым (:focus-visible + «страховка» на :focus).
  2. Иконке добавьте aria-label.
Редактор решения
Самостоятельно: Добавьте вторую иконку-кнопку «Поиск» с тем же стилем фокуса.

Дополнительно

  • Проверьте контраст текста на кнопке ≥ 4.5:1.
  • Добавьте title к иконкам как подсказку для мыши.
